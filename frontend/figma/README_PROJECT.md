# InCollab - Платформа поиска участников для проектов

> Современная веб-платформа для студентов и стартапов, помогающая находить участников для проектов и компаниям находить талантливых исполнителей.

## 🎯 Основные функции

### Для студентов и создателей проектов:
- ✅ Создание и управление проектами
- ✅ Публикация вакансий с требованиями по навыкам
- ✅ Получение откликов от кандидатов
- ✅ Рекомендации подходящих участников
- ✅ Управление командой проекта

### Для соискателей:
- ✅ Персонализированная лента проектов
- ✅ Фильтрация по навыкам и интересам
- ✅ Отклики на интересные позиции
- ✅ Профиль с портфолио навыков

### Система подписок:
- **Free**: 2 проекта, 3 вакансии, 10 просмотров кандидатов
- **Pro**: 10 проектов, 10 вакансий, 100 просмотров + аналитика
- **Enterprise**: Без ограничений + API доступ

## 🚀 Технологический стек

- **Frontend**: React 18 + TypeScript
- **Routing**: React Router v7 (Data Mode)
- **Styling**: Tailwind CSS v4
- **UI Components**: Radix UI + shadcn/ui
- **Forms**: React Hook Form
- **Icons**: Lucide React
- **Notifications**: Sonner
- **Animations**: Motion (Framer Motion)
- **Date handling**: date-fns
- **Backend**: FastAPI (готов к интеграции)

## 🎨 Основные страницы

### 1. Лента проектов (/)
- Список всех активных проектов
- Поиск по названию и описанию
- Фильтрация по навыкам
- Вкладки: Все / Рекомендации / Новые

### 2. Детали проекта (/projects/:id)
- Полная информация о проекте
- Список вакансий
- Состав команды
- Форма отклика на позицию

### 3. Создание проекта (/create-project)
- Форма с описанием и целями
- Добавление тегов
- Указание текущего прогресса

### 4. Мои проекты (/my-projects)
- Созданные проекты
- Проекты, в которых участвую
- Мои отклики и их статусы

### 5. Профиль (/profile)
- Информация о пользователе
- Список навыков
- История проектов

### 6. Настройки (/settings)
- Редактирование профиля
- Управление навыками
- Настройки приватности
- Выбор подписки

### 7. Аутентификация
- Вход (/login)
- Регистрация (/register)
- OAuth (Google, GitHub)

## 🔌 Интеграция с API

Приложение подготовлено для интеграции с FastAPI backend.

**Текущее состояние**: Работает на моковых данных
**Готовые endpoints**: Auth, Users
**Требуется реализация**: Projects, Positions, Applications, Skills

Подробное руководство: [API_INTEGRATION.md](./API_INTEGRATION.md)

### Быстрый старт с API

1. Установите переменные окружения:
```bash
echo "VITE_API_BASE_URL=http://localhost:8000" > .env
```

2. Раскомментируйте реальные API вызовы в `LoginPage.tsx` и `RegisterPage.tsx`

3. Реализуйте недостающие endpoints на backend

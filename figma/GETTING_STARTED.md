# Быстрый старт с InCollab

## 🎯 Что это?

**InCollab** - это платформа для студентов и стартаперов, которая помогает:
- Находить участников для учебных и стартап-проектов
- Откликаться на интересные вакансии в проектах
- Создавать команды мечты
- Компаниям находить талантливых исполнителей

## 🚀 Первые шаги

### Для новых пользователей

1. **Регистрация**
   - Перейдите на `/register`
   - Заполните форму (email, имя пользователя, пароль)
   - Или используйте OAuth (Google/GitHub)
   - **Для демо**: используйте `demo / demo` на странице `/login`

2. **Настройка профиля**
   - Перейдите в "Настройки" (Settings)
   - Добавьте информацию о себе
   - Укажите свои навыки - это важно для персональных рекомендаций!
   - Настройте приватность профиля

3. **Изучите проекты**
   - На главной странице вы увидите ленту проектов
   - Используйте поиск и фильтры по навыкам
   - Просмотрите вкладку "Рекомендации" - там проекты, подобранные специально для вас

### Для создателей проектов

1. **Создайте проект**
   - Нажмите "Создать проект" в header
   - Заполните:
     - Название проекта
     - Подробное описание
     - Цели и задачи
     - Текущий прогресс
     - Теги для поиска

2. **Добавьте вакансии**
   - Откройте свой проект
   - Нажмите "Добавить вакансию"
   - Укажите:
     - Название позиции (Frontend Dev, Designer и т.д.)
     - Описание обязанностей
     - Требуемые навыки
   
3. **Получайте отклики**
   - Кандидаты будут откликаться на ваши вакансии
   - Просматривайте их профили и навыки
   - Принимайте или отклоняйте заявки
   - Формируйте команду!

### Для соискателей

1. **Найдите интересный проект**
   - Просмотрите ленту проектов
   - Используйте фильтры по навыкам
   - Читайте описания и цели проектов

2. **Откликнитесь на вакансию**
   - Откройте детали проекта
   - Перейдите во вкладку "Вакансии"
   - Нажмите "Откликнуться"
   - Напишите сопроводительное сообщение

3. **Отслеживайте статус**
   - Перейдите в "Мои проекты" → "Мои отклики"
   - Смотрите статус: На рассмотрении / Принят / Отклонен

## 📋 Основные разделы

### 🏠 Главная (/)
Лента всех проектов с поиском и фильтрами
- **Все проекты** - полный список
- **Рекомендации** - подобрано на основе ваших навыков
- **Новые** - недавно добавленные

### 💼 Мои проекты (/my-projects)
Управление вашими проектами
- **Мои проекты** - созданные вами
- **Участвую** - проекты, где вы член команды
- **Мои отклики** - статус ваших заявок

### ➕ Создать проект (/create-project)
Форма создания нового проекта

### 👤 Профиль (/profile)
Ваш публичный профиль
- Информация о вас
- Навыки
- Проекты

### ⚙️ Настройки (/settings)
- **Профиль** - редактирование данных
- **Навыки** - управление списком навыков
- **Приватность** - настройки видимости
- **Подписка** - выбор тарифного плана

## 💡 Полезные советы

### Для создателей проектов

✅ **Подробно описывайте проект**
- Чем детальнее описание, тем больше откликов
- Укажите реальные цели и текущее состояние

✅ **Правильно подбирайте навыки**
- Указывайте только необходимые навыки
- Не перегружайте требованиями

✅ **Будьте активны**
- Быстро отвечайте на отклики
- Обновляйте статус проекта

✅ **Используйте теги**
- Добавляйте релевантные теги (#EdTech, #AI, #Mobile)
- Это поможет людям найти ваш проект

### Для соискателей

✅ **Заполните профиль**
- Добавьте фото
- Опишите свой опыт
- Укажите все релевантные навыки

✅ **Персонализируйте отклики**
- Не отправляйте одинаковые сообщения
- Объясните, почему вам интересен именно этот проект
- Расскажите о релевантном опыте

✅ **Будьте честны**
- Указывайте реальный уровень навыков
- Не приукрашивайте опыт

## 🎓 Тарифные планы

### Free (Бесплатно)
- 2 проекта
- 3 вакансии на проект
- 10 просмотров кандидатов/месяц
- Базовая поддержка

### Pro ($9/месяц)
- 10 проектов
- 10 вакансий на проект
- 100 просмотров кандидатов/месяц
- Приоритетная поддержка
- Аналитика проектов

### Enterprise ($29/месяц)
- Неограниченно проектов
- Неограниченно вакансий
- Неограниченно просмотров
- Персональный менеджер
- API доступ

## 🔐 Безопасность

- Все данные защищены шифрованием
- Используется JWT аутентификация
- Вы можете сделать профиль приватным
- Мы не передаем данные третьим лицам

## ❓ Частые вопросы

**Как удалить проект?**
Перейдите в детали проекта → Настройки → Удалить проект

**Можно ли редактировать проект после создания?**
Да, нажмите "Редактировать" в карточке проекта

**Как найти проекты под мои навыки?**
Заполните навыки в настройках и смотрите вкладку "Рекомендации"

**Сколько стоит использование?**
Базовый план полностью бесплатный. Pro и Enterprise - платные.

**Можно ли участвовать в нескольких проектах?**
Да, ограничений нет!

**Как связаться с владельцем проекта?**
Через отклик на вакансию или чат (в разработке)
